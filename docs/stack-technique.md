# Stack technique

| Couche      | Technologie                      | Justification                               |
| ----------- | -------------------------------- | ------------------------------------------- |
| Framework   | **Next.js 14+ (App Router)**     | SSG, excellent SEO, composants React        |
| UI          | **React + TypeScript**           | Typage strict, composants reutilisables     |
| Styling     | **Tailwind CSS v4 + shadcn/ui**  | Design system coherent, mobile-first        |
| Animations  | **GSAP + Three.js**              | Hero parallax moto, micro-interactions      |
| CMS         | **Pas en V1** (pages statiques)  | Sanity.io en Phase 2 si besoin              |
| Formulaires | **React Hook Form + Resend**     | Contact, devis, prise de RDV                |
| Hebergement | **Vercel**                       | Deploy auto, CDN global, analytics          |
| SEO         | **next-sitemap + meta par page** | Correction de tous les problemes identifies |
