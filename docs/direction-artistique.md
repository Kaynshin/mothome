## Hero concept : "Anatomy of Speed"

Une moto se decompose en scrollant en mode parallax pour reveler les composants.

### Sequence de scroll

1. **Scroll 0%** : moto entiere, sombre, cinematique
2. **Scroll 25-75%** : decomposition progressive par calques (moteur, freins, transmission...)
3. Chaque composant flotte avec un label lie aux services Mothome
4. **Scroll 100%** : re-assemblage avec CTA principal

### Implementation technique

- Three.js + GSAP ScrollTrigger
- Modele 3D segmente (~100 EUR sur Sketchfab)
- Estimation : 3-5 jours de dev frontend
