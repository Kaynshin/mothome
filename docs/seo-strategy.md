## 3. Strategie de liens internes

### Principes
- Chaque page doit avoir au moins 2 liens entrants depuis d'autres pages du site
- Le maillage suit la logique du parcours utilisateur (decouverte → service → conversion)
- Les CTA sont des liens internes deguises — chaque CTA pointe vers une page cible

### Matrice de maillage interne

| Page source | Liens vers | Ancre type |
|---|---|---|
| Accueil | L'Atelier, Service a domicile, Le Bar, Contact | CTA services, "Decouvrir l'atelier", "Prendre RDV" |
| L'Atelier | Service a domicile, Accessoires, Contact | "Aussi a domicile", "Nos accessoires", "Demander un devis" |
| Service a domicile | L'Atelier, Contact | "Voir nos tarifs atelier", "Reserver une intervention" |
| Accessoires | L'Atelier, Depot-Vente | "Entretien de votre equipement", "Motos disponibles" |
| Depot-Vente | L'Atelier, Contact | "Controle mecanique inclus", "Nous contacter" |
| Le Bar | Accueil, A propos | "Le garage", "Notre histoire" |
| A propos | L'Atelier, Le Bar, Contact | "Nos services", "Le bar", "Nous rencontrer" |
| Contact | L'Atelier, Service a domicile | "Nos services", "Intervention a domicile" |

### Pages piliers et pages satellites
- **Page pilier** : L'Atelier (concentre l'autorite SEO service/reparation)
- **Pages satellites** : Service a domicile, Accessoires (renforcent la pilier par liens internes)
- **Page conversion** : Contact (cible finale de tous les parcours)

## 5. Checklist SEO technique (a transmettre au Dev Back-End)

### Performance / Core Web Vitals
- [ ] LCP < 2.5s sur toutes les pages (images WebP/AVIF, lazy loading, CDN)
- [ ] CLS < 0.1 (dimensions explicites sur images/videos, pas de layout shift)
- [ ] INP < 200ms (minimiser JS bloquant, defer scripts non critiques)
- [ ] Compression Brotli sur le serveur
- [ ] Cache HTTP agressif sur les assets statiques (immutable, max-age 1 an)

### Indexation et crawl
- [ ] Sitemap XML automatique (next-sitemap) — soumettre a Google Search Console
- [ ] robots.txt valide (pas de blocage accidentel des assets)
- [ ] Balise canonical sur chaque page
- [ ] Pas de pages orphelines (chaque page atteignable en 3 clics max)
- [ ] Pagination correcte si listing motos depot-vente > 1 page

### Balises HTML
- [ ] Meta title unique par page (50-60 caracteres)
- [ ] Meta description unique par page (140-160 caracteres)
- [ ] Une seule balise H1 par page
- [ ] Hierarchie Hn logique (H1 > H2 > H3, pas de saut)
- [ ] Alt text descriptif sur toutes les images (inclure mots-cles naturellement)
- [ ] Balise lang="fr" sur le html

### Donnees structurees
- [ ] JSON-LD MotorcycleRepair sur toutes les pages (schema principal)
- [ ] Schema Service sur L'Atelier et Service a domicile
- [ ] Schema Product sur Accessoires et Depot-Vente
- [ ] Schema BarOrPub sur Le Bar
- [ ] Valider tous les schemas avec Google Rich Results Test avant MEP

### URLs et navigation
- [ ] URLs propres, en minuscules, avec tirets (ex: /l-atelier, /service-a-domicile)
- [ ] Pas de trailing slash inconsistant
- [ ] Redirections 301 de toutes les anciennes URLs Joomla vers les nouvelles
- [ ] Breadcrumb avec BreadcrumbList schema sur chaque page

### Mobile et accessibilite
- [ ] Design mobile-first responsive
- [ ] Boutons CTA min 44x44px (zone de tap)
- [ ] Texte lisible sans zoom (min 16px corps)
- [ ] Contraste suffisant (ratio 4.5:1 minimum — attention au theme dark)

### Securite et HTTPS
- [ ] HTTPS partout, redirection HTTP → HTTPS
- [ ] Headers de securite (HSTS, X-Content-Type-Options, etc.)

### Google Business Profile
- [ ] Fiche optimisee : nom, adresse, telephone, horaires, categorie "Reparation de motos"
- [ ] Photos du garage, de l'atelier et du bar
- [ ] Lien vers le site web mothome.fr
- [ ] Categorie secondaire : "Bar"
- [ ] Activer les posts Google et les Q&A

## 6. Analyse concurrentielle locale

### Concurrents identifies a Thonon-les-Bains

| Concurrent | Positionnement | Forces | Faiblesses vs Mothome |
|---|---|---|---|
| FK Moto | Multi-marques, reparation | Large spectre marques | Pas de bar, pas de concept unique |
| Profil Motos Thonon | Concessionnaire Yamaha | Marque officielle, show room | Monoemarque, pas d'artisanal |
| RV Moto | Garage classique | Presence locale | Site basique, pas de differenciateur |
| Kick Moto | Honda/Vespa/Piaggio | 600m2 show room | Concessionnaire, pas artisanal |
| Dafy Speed | Equipement/accessoires | Marque nationale | Pas de reparation, pas local |

### Avantage concurrentiel Mothome
Le concept hybride garage + bar est **unique sur le marche local**. Aucun concurrent ne propose cet espace de vie. C'est le principal axe de differenciation a pousser dans le contenu et le SEO.
